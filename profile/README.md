## ⛈️ **패션과 날씨,이제** 웨더웨어**에서 한 번에!**

**웨더웨어**는 **날씨를 기반으로 개인 맞춤형 옷차림 추천 서비스를 제공하는 패션 커뮤니티**입니다.

사용자들과 함께 OOTD(outfit of the day, 오늘의 옷차림) 패션 스타일을 공유하고,
오늘의 날씨에 딱 맞는 나만의 특별한 OOTD로 스타일리시하게 하루를 시작해보세요!
